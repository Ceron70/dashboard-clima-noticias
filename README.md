# 🌤️ Dashboard del Clima + Noticias

Aplicación full-stack que muestra el clima actual y noticias de cualquier ciudad del mundo, guardando el historial de búsquedas en una base de datos PostgreSQL.

## 🚀 Tecnologías utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

### Backend
- Node.js
- Express
- PostgreSQL

### APIs externas
- OpenWeatherMap API (clima)
- NewsData.io API (noticias)

## 📋 Funcionalidades

- ✅ Buscar clima por nombre de ciudad
- ✅ Mostrar temperatura, humedad, viento y descripción del clima
- ✅ Mostrar noticias relacionadas al país
- ✅ Guardar historial de búsquedas en PostgreSQL
- ✅ Diseño responsive
- ✅ Interfaz amigable y moderna
